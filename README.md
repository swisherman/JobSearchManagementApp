# Job Search Management Application

A Blazor Server application for managing job-search activities, applications, companies, projects, and related files.

## Why This Exists

Managing a job search often involves tracking companies, applications, follow-up dates, projects, resumes, and supporting documents across multiple systems.

This application centralizes those activities into a single dashboard backed by MongoDB.

## Features

* Company management
* Job posting management
* Application tracking
* Project tracking
* File management
* Dashboard metrics
* Interview tracking

## Tech Stack

* C#
* ASP.NET Core
* Blazor Server
* MongoDB
* MudBlazor
